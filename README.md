# Swiggy Clone - Java Backend

This is a backend clone of Swiggy built using Java and Spring Boot.

## Features
- User registration & login
- Restaurant listing
- Cart & order management
- JWT-based authentication

## Tech Stack
- Java 17
- Spring Boot
- MySQL
- JPA / Hibernate
- Maven
